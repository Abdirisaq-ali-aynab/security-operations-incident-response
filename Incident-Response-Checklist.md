# Incident Response Checklist

## Detection & Triage
- [x] Alert reviewed and validated  
- [x] Indicators of compromise identified  
- [x] Severity and priority assessed  

## Investigation
- [x] Endpoint activity analyzed  
- [x] Logs reviewed for suspicious behavior  
- [x] Persistence mechanisms identified  

## Containment
- [x] Affected system isolated  
- [x] Credential hygiene actions recommended  

## Eradication & Recovery
- [x] Malicious artifacts removed  
- [x] System integrity validated  

## Post-Incident Review
- [x] Control gaps identified  
- [x] Lessons learned documented  
- [x] Preventive recommendations created
